# 📱 Mobile Sales Dashboard (Power BI)

## 📌 Project Overview

The **Mobile Sales Dashboard** is an interactive Power BI dashboard developed to analyze mobile phone sales across different cities, brands, models, payment methods, and time periods. It provides key business insights through dynamic visualizations, KPI cards, maps, charts, and slicers that help users monitor sales performance and customer purchasing behavior.

---

## 🚀 Features

- 📊 Interactive Power BI Dashboard
- 💰 Total Sales Analysis
- 📦 Total Quantity Sold
- 🧾 Total Transactions
- 📈 Average Sales Value
- 🗺️ Sales Distribution by City (Map Visualization)
- 📅 Quantity Sold by Day
- ⭐ Customer Ratings Analysis
- 💳 Payment Method Distribution
- 📱 Mobile Model-wise Sales Analysis
- 🏷️ Brand-wise Sales Summary
- 📅 Month-wise Navigation Panel
- 🔍 Dynamic Filters for:
  - Mobile Model
  - Brand
  - Payment Method
  - Day Name
  - Month Selection

---

## 📈 Dashboard KPIs

The dashboard tracks the following Key Performance Indicators (KPIs):

- **Total Sales:** 3M
- **Total Quantity Sold:** 66
- **Total Transactions:** 12
- **Average Sales:** 46K
